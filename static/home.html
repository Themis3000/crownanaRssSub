Welcome to the Crownana email notification service!

Turn your personal website's existing RSS feed into an email subscription with ease! Simply copy + paste the widget to your webpage & fill in your rss feed URL. Style it to your hearts content!

Features

  • Email verification at signup
  • 1 click unsubscribe
  • Maximum notification frequency controls
    • Ensures your visitors won't get more emails than they want from you.
    • Can be changed at any time by the user.

All these features are in place to ensure your users won't see your update emails as a nuisance down the road.

How to host on your own

Prefer to run it on your own? You can grab the source code from github.com/Themis3000/crownanaRssSub

I use mailgun as my email service provider (which provides 100 free emails a day). If you prefer not to use another email service provider, it's quite easy to adapt the program to sending emails via any other method too if you have some programming experience!

Here's the instructions for a linux system:

  1. Set up an account on mailgun and get an api key (and connect your domain, if you want)
  2. Install docker and docker compose
  3. Create and cd into a directory you would like install the server into
  4. Run git clone https://github.com/Themis3000/crownanaRssSub.git .
  5. Create your docker compose configuration
    1. Copy the example compose file with cp ./docker-compose-example.yml ./docker-compose.yml
    2. Edit the compose file with nano ./docker-compose.yml
    3. Fill in a database password. Make sure you use the same one in both places!
    4. Fill in the url of the domain you will be hosting your rss feed site on
    5. Fill in the email address you'll be sending from
    6. Fill in your "service name". This will be displayed in the footer of emails
    7. Fill in your mailgun api key
  6. Run docker compose up -d to start it up
  7. Visit http://localhost to verify that it's running correctly. At this point it should fully function!
  8. When you want to shut it down, run docker compose down
